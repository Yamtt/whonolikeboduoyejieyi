* ext4 ：可以扩展磁盘空间大小
* squashfs ：可以使用 重置功能（恢复出厂设置）
* efi : efi引导，非BIOS引导（优先使用efi固件，无法启动时再换无efi固件）   
* rootfs ：不带引导，可自行定义用grub或者syslinux来引导      
* combined ：带引导 
* .img ：物理机使用
* .vmdk ：虚拟机 VMware 使用
* .qcow2 ：虚拟机 PVE 使用
* .vdi ：虚拟机 VirtualBox 使用
* .vhdx ：虚拟机 Hyper-V 使用
